# How to learn Plus

Первый проект в рамках прохождения курса "Веб-разработчик плюс" от Яндекс Практикум.

Чтобы успешно пройти ревью, проект должен соответствовать специальным [чек-листам](https://code.s3.yandex.net/web-plus/checklists/checklist-1/index.html)

[Стандарт по оформлению кода](https://code.s3.yandex.net/web-developer/static/design-rules/index.html)

# Briefs

[Первый этап](https://code.s3.yandex.net/web-plus/project-1/sprint-1-brief.pdf) — эту часть проекта вы осилите, когда изучите флексбокс, позиционирование, семантику и БЭМ.

[Второй этап](https://code.s3.yandex.net/web-plus/project-1/sprint-2-brief.pdf) — эту часть доработки проекта проделайте, когда изучите анимации с трансформациями и работу со встроенными элементами.
